Use the ne_countries function to load European countries in the europe dataframe. Pay attention that returnclass is "sf", and that the map is at scale "large". Filter out Russia, using the normal dplyr::filter function.

europe <- ne_countries(scale = 'large', type ="countries", continent = "europe", returnclass = "sf") %>%
filter(sovereignt != "Russia")

Plot the europe dataset using geom_sf.

  geom_sf()

Compute for each country the GDP per capita, using data from the columns gdp_md_est and pop_est. Start from the cropped dataset of the previous point. Then create a new plot, with the resulting variable as the fill aesthetic. Also, play with the size and color parameters of geom_sf to adjust the appearance of borders between countries.

cropped %>%
  select(gdp_md_est, pop_est, sovereignt) %>%
 mutate("gdp_capita" = (gdp_md_est /as.numeric(pop_est))*1000000) %>%
ggplot(aes(fill = gdp_md_est)) + 
geom_sf()
